# MainConstructions
Итоговый проект по Блоку «Основные конструкции C++»

Создано консольное приложение «Чат». Название проекта – MainConstructions.
Идея решения – использование двух классов, реализующих работу с массивом пользователей чата и массивом сообщений в чате. В каждом классе реализуются функции по работе с массивами – добавление записей, проверка вводимой информации, получение содержимого. Написаны две процедуры – редактор пользователей и редактор чата, которые используют экземпляры соответствующих классов.

В головном модуле реализован цикл со считыванием символа от пользователя. В зависимости от введенного символа вызывается процедура работы со списком пользователей, процедура работы с чатом или закрытие программы.

Для работы со списком пользователей создан класс Users, который содержит массив из объектов класса User. Класс User имеет приватные поля, содержащие id, имя, логин и пароль пользователя. Класс Users помимо массива объектов класса User содержит информацию о состоянии массива и реализует различные функции работы со списком пользователей – добавление пользователя, получение информации о заполненности массива, поиск и получение данных о пользователе, авторизация пользователя.

Для работы с чатом создан класс Chat, который содержит динамические массивы на базе шаблонного класса Array и функции добавления и чтения сообщений (публичных и приватных). После успешной авторизации пользователя появляется возможность отправки сообщений от имени текущего пользователя в публичный или приватный чат с проверкой корректности указания адресата сообщения. Реализовано чтение всех сообщений из публичного чата и всех сообщений из приватного чата, которые имеют в качестве отправителя или получателя только текущего пользователя.

Функция регистрации пользователя реализована в CallUsersEditor .
Функция входа в чат по логиу/паролю реализована в Login класса Users.
Функции отправки сообщений и обмена сообщений реализованы в функциях класса Chat.
Дополнительно реализована обработка исключений при выходе заданного значения за допустимые пределы и использован шаблонный класс Array.
